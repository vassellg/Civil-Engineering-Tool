# Civil Engineering Tools

A collection of Python scripts and CAD/BIM projects built as part of 
my civil engineering portfolio.

## Soil Classification Tool
Classifies soils using the USCS system (ASTM D2487) based on sieve 
analysis and Atterberg limit inputs. Outputs classification symbol, 
name, and auto-generates a Casagrande plasticity chart.

**Libraries:** Python, matplotlib, numpy
**Topics:** Geotechnical Engineering, Soil Mechanics
